# antigravity.md: Project Constitution

## 1. プロジェクト概要
- **プロジェクト名**: 来世は猫 (Next Life: Cat)
- **目的**: プレミアムキャットフードの販売LP制作
- **ターゲット**: 猫を飼っている、猫を家族同然に愛する人々
- **デザイン**: シンプル、清潔感、猫の写真ファースト（可愛さ重視）
- **キャッチコピー**: 「来世は猫になりたい。」それくらい、愛おしい毎日を。

## 2. 技術スタック
- **Core**: React (Vite)
- **Styling**: Vanilla CSS (CSS Variables)
- **Deploy**: Vercel (予定)
- **Assets**: ユーザー提供または生成された猫の画像

## 3. MVP（最小構成）スコープ
1. **Hero Section**: 画面いっぱいの可愛い猫の写真とメッセージ
2. **Concept**: 「体に優しい」ことを伝えるストーリー
3. **Features**: 健康へのこだわり（3つのポイント）
4. **CTA**: 購入ボタン（シンプルに）

- プロジェクト初期化完了
